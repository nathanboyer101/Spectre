// Comment ça fonctionne

Simple.
Puissant. Immédiat.

Installez l'extension, ouvrez n'importe quelle page — Spectre fait le reste automatiquement.

01

Installation en 1 clic

Ajoutez Spectre à votre navigateur depuis le Chrome Web Store ou Firefox Add-ons. Aucune configuration requise, aucun compte à créer.

02

Détection intelligente

Spectre identifie automatiquement chaque espace publicitaire sur les pages que vous visitez, quelle que soit leur taille ou leur format.

03

Transformation instantanée

Les publicités sont remplacées en temps réel par les images de votre choix. William, paysages, animaux… à vous de décider.
