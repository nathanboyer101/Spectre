# 👻 Spectre

**Willi Blocker** est une extension de navigateur open-source développée par **Spectre**, un groupe de passionnés souhaitant rendre le web plus propre, plus rapide et plus agréable à utiliser.

Elle bloque automatiquement les publicités, remplace les espaces publicitaires par du contenu visuel personnalisé et tente de contourner certaines détections anti-adblock afin d'améliorer l'expérience de navigation.

## ✨ Fonctionnalités

- 🚫 Blocage intelligent des publicités
- 🖼️ Remplacement des espaces publicitaires par des visuels personnalisés
- 🧹 Suppression des contenus sponsorisés et éléments promotionnels
- 🛡️ Réduction des principaux traqueurs publicitaires
- ⚡ Léger, rapide et optimisé
- 🔓 Entièrement open-source
- 🦊 Disponible officiellement sur Firefox Add-ons

## 📦 Installation

### 🦊 Firefox (recommandé)

Installez directement Willi Blocker depuis le catalogue officiel Mozilla :

[![Firefox Add-ons](https://blog.mozilla.org/addons/files/2015/11/get-the-addon.png)](https://addons.mozilla.org/fr/firefox/addon/williblocker/)

---

### Chromium (Chrome, Edge, Brave, Opera...)

L'extension n'est pas encore publiée sur les stores Chromium.

En attendant :

1. Clonez / Téléchargez ce dépôt / Téléchargez le .zip depuis notre site
2. Ouvrez la page des extensions :
   - Chrome : `chrome://extensions`
   - Edge : `edge://extensions`
   - Brave : `brave://extensions`
3. Activez le **Mode développeur**.
4. Cliquez sur **Charger l'extension non empaquetée**.
5. Sélectionnez le dossier du projet.

## ⚙️ Comment ça fonctionne ?

Spectre détecte les publicités grâce à plusieurs techniques complémentaires :

- Analyse des domaines publicitaires connus
- Détection des classes et attributs utilisés par les régies
- Surveillance dynamique du DOM
- Remplacement ou suppression des éléments détectés

L'objectif est de conserver une mise en page propre tout en supprimant les contenus publicitaires.

## 🔒 Confidentialité

Votre vie privée est une priorité.

Spectre :

- ✅ Ne collecte aucune donnée personnelle
- ✅ N'effectue aucun suivi de navigation
- ✅ N'envoie aucune information à un serveur distant

Tout le traitement est réalisé localement dans votre navigateur.
