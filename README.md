# 👻 Spectre

**Willi Blocker** est une extension de navigateur open-source conçue pour rendre le web plus propre et plus agréable à utiliser.

Elle bloque automatiquement les publicités, remplace les espaces publicitaires par du contenu visuel personnalisé et tente de contourner certaines détections anti-adblock afin d'améliorer l'expérience de navigation.

## ✨ Fonctionnalités

- 🚫 Blocage des publicités classiques
- 🖼️ Remplacement des emplacements publicitaires par des images personnalisées
- 🧹 Suppression des bannières et éléments sponsorisés
- 🛡️ Réduction des traqueurs publicitaires courants
- ⚡ Léger et rapide
- 🔓 Projet open-source

## 📦 Installation

### Firefox

1. Téléchargez l'archive ZIP depuis la page des releases.
2. Ouvrez `about:debugging`.
3. Cliquez sur **This Firefox**.
4. Sélectionnez **Load Temporary Add-on**.
5. Choisissez le fichier `manifest.json`.

### Chrome / Edge / Brave

1. Téléchargez le projet.
2. Ouvrez la page des extensions :
   - Chrome : `chrome://extensions`
   - Edge : `edge://extensions`
3. Activez le **Mode développeur**.
4. Cliquez sur **Charger l'extension non empaquetée**.
5. Sélectionnez le dossier du projet.

## ⚙️ Comment ça fonctionne ?

Spectre détecte les éléments publicitaires grâce à plusieurs techniques :

- Analyse des domaines publicitaires connus
- Détection de classes et attributs fréquemment utilisés par les régies
- Surveillance dynamique des nouveaux éléments ajoutés à la page
- Remplacement visuel des publicités détectées

Les éléments publicitaires sont ensuite masqués ou remplacés automatiquement.

## 🔒 Confidentialité

Spectre :

- Ne collecte aucune donnée utilisateur
- N'envoie aucune information à un serveur externe
- Ne réalise aucun suivi de navigation

Tout le traitement est effectué localement dans votre navigateur.
